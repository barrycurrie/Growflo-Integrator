Growflo API

Users

Users are the backbone of interacting with the Growflo system.Without a user account you cannot interact with the interface or through the API. Note: API tokens are only returned for the current authenticated user.

Get all Users

Returns all the registered users within the Growflo system.

REQUEST

GET

/users{?include}
Parameters
include string roles,permissions

Allows you include the following information in the response:

  • roles

  • permissions

RESPONSE

Headers
Content-Type application/json
200 application/json
BODY SCHEMA 
                        {
    "data": [
        {
            "id": 1,

User Permissions

Permissions are used to control access to the interface and API. They can be created and updated through the API but they have to be implemented in the system for changes to take effect. We recommend that you do not use the create/store/delete endpoints.

REQUEST

GET

  • roles

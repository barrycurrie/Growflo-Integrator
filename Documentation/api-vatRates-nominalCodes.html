Settings - VAT Rates

These endpoints contain all the details about VAT Rates that exist within the system.Customers and Stock lines are assigned to VAT Rates to work out correct pricing.

Get all VAT Rates

REQUEST

GET

/vat_rates

RESPONSE

Headers
Content-Type application/json
200 application/json
BODY SCHEMA 
                        {
    "data": [
        {
            "id": 1,
            "code": "T0",
            "rate": "0.00"
        },
        {
            "id": 2,
            "code": "1",
            "rate": "20.00"
        },
        {
            "id": 3,
            "code": "4",
            "rate": "0.00"
        }
    ]
}

Get a specific Vat Rate

REQUEST

GET

/vat_rates/{vat_rate}
Parameters
vat_rate integer 1

The ID of the VAT Rate

RESPONSE

Headers
Content-Type application/json
200 application/json
BODY SCHEMA 
                        {
    "data": {
        "id": 1,
        "code": "T0",
        "rate": "0.00"
    }
}

Create a VAT Rate

REQUEST

POST

/vat_rates
Headers
Content-Type application/json
application/json
BODY SCHEMA 
                          {
    "code": "T1",
    "rate": 10
}

RESPONSE

Headers
Content-Type application/json
200 application/json
BODY SCHEMA 
                        {
    "data": {
        "id": 2,
        "code": "T1",
        "rate": "10.00"
    }
}

Update a VAT Rate

REQUEST

PUT

/vat_rates/{vat_rate}
Parameters
vat_rate integer 2

The ID of the VAT Rate you want to update
Headers
Content-Type application/json
application/json
BODY SCHEMA 
                          {
    "code": "T1 Update",
    "rate": 10
}

RESPONSE

Headers
Content-Type application/json
200 application/json
BODY SCHEMA 
                        {
    "data": {
        "id": 2,
        "code": "T1 Update",
        "rate": "10.00"
    }
}

Delete a VAT Rate

REQUEST

DELETE

/vat_rates/{vat_rate}
Parameters
vat_rate integer 1

The ID of the VAT Rate

RESPONSE

Headers
Content-Type application/json
200 application/json
BODY SCHEMA 
                        {
    "deleted": true
}
